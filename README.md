# Spray Tracker

Utility-first iOS-style Flutter app for tracking vegetable garden spray activity.

## MVP direction

The first version focuses on the core utility:

- Quickly log a spray application
- See what is safe or not safe to harvest
- Review recent spray activity
- Track withholding periods
- Keep the dashboard simple and mobile-first

Garden beds will be added manually later as a dedicated management feature. The MVP does not depend on bed setup.

## Current design language

- iOS-style cards
- Large dashboard typography
- Soft grouped sections
- Bottom tab navigation
- Clear safety state labels
- Minimal form friction
